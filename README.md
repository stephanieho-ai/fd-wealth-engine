# FD Wealth Engine 💰

A smart Fixed Deposit portfolio system built to help you track, plan, and optimize your FD investments.

## ✨ Features
- 📊 Track your Fixed Deposits
- 📅 Plan maturity timelines
- 💡 Optimize interest strategy
- 🏦 Multi-bank support

## 🛠 Tech Stack
- React
- Vite
- Electron

## 🚀 How to Run
npm install
npm run dev
